# Workspace Archive

This folder holds historical workspace-level artifacts that should not remain in the top-level directory.

## Purpose

- Reduce noise in the workspace root.
- Preserve older patch files, migration notes, and one-off workspace artifacts without presenting them as active operational files.
- Keep the root focused on current repos, entry docs, and deployment helpers.

## Structure

- `root-history/` stores former root-level artifacts that are no longer active workspace entry points.

Archive contents are reference material only and should not be treated as current instructions unless separately validated.
