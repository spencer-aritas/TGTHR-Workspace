# Archive

This folder holds historical implementation notes, fix logs, and one-off status reports that are no longer the primary entry points for the PWA repository.

## Structure

- `root-history/`: files previously kept at the repository root during implementation and troubleshooting bursts

## Keep at Repo Root

Keep only active entry-point documents at the repository root, such as:
- `README.md`
- `QUICK_REFERENCE.md`
- `DEBUG_GUIDE.md`
- `SECURITY.md`
- any actively maintained operator or deployment references

## Archive Policy

Archive a document here when it is primarily historical, retrospective, or tied to a one-time fix or implementation phase. If the document still contains durable operational guidance, summarize that guidance into an active README or runbook before archiving it.
